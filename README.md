# VDOO-attack-code
The provided code accompany the paper Breaking the Spell and generates an instance of of a VDOO public key then find a vector in O2 using the simple attack combined with the Minrank attack. 

To use this code, copy and paste the text from [keyandattack.m](https://github.com/antsaaims/VDOO-attack-code/blob/main/keyandattack.m) to the [Magma Computer Algebra Web interface](http://magma.maths.usyd.edu.au/calc/).
